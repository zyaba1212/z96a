## Хронология записей

### Защиты Leaflet и возврат на глобус (сессия 2025-02-20)

- **Что сделано:** в `EarthScene.tsx` добавлены защиты при уничтожении карты Leaflet и возврате на глобус: `destroyingMapRef`, `safeMapCall`, `isMapContainerInDoc`. В `returnToGlobe`: отключение `scrollWheelZoom`, `map.off()`, обнуление ref до `remove()`, проверка контейнера в DOM перед `remove()`. В обработчиках zoomend/moveend — проверки и try/catch. Overlay остаётся в DOM (visibility/pointerEvents). Cleanup с проверкой контейнера; в эффекте Leaflet — флаг `cancelled`, очистка на верхнем уровне, перед `L.map` удаление старой карты по ref.
- **Ошибка:** `TypeError: Cannot read properties of undefined (reading '_leaflet_pos')` в Leaflet при переходе обратно на глобус (стек: `getPosition` → `_getMapPanePos` → `_getNewPixelOrigin` → `_move` → `_onZoomTransitionEnd`).
- **Причина:** колбэк Leaflet `_onZoomTransitionEnd` (окончание анимации зума) срабатывал уже после удаления карты (двойной `requestAnimationFrame` давал ~32 ms, анимация зума в Leaflet длится дольше).
- **Решение:** заменить двойной rAF перед `map.remove()` на `setTimeout(..., 400)` в двух местах — в `returnToGlobe` и в cleanup эффекта Leaflet. За 400 ms Leaflet успевает отработать `_onZoomTransitionEnd` по ещё существующему DOM.

### Порог перехода в 2D и гистерезис (2025-02-20)

- **Что сделано:** порог входа в 2D задан константой `MAP_THRESHOLD_ENTER_2D = 1.2`; переход при `dist <= MAP_THRESHOLD_ENTER_2D`. Введён `transitionTo2DScheduledRef`, чтобы не планировать переход повторно; сброс при возврате на глобус. ResizeObserver с debounce (таймер 60 ms) для уменьшения «дёргания» кнопок при ресайзе.
- **Ошибка (ранее):** кнопки «слетали» при зуме колёсиком; кнопка «На карту» пропадала после «На глобус».
- **Решение:** layout корня 100vw/100vh, `#z96a-ui-root` fixed с явными размерами; UI вынесен из overflow. Кнопка «На карту» всегда показывается на глобусе; порядок на глобусе: сначала «−», потом «+». Двойной rAF при входе в 2D, один rAF в `returnToGlobe` перед setState.

### «Map container is already initialized» (2025-02-20)

- **Ошибка:** Leaflet выдавал ошибку при повторной инициализации контейнера.
- **Решение:** в эффекте Leaflet — флаг `cancelled`, cleanup в начале; перед вызовом `L.map` проверять и удалять существующую карту по `mapInstanceRef.current`.

### Подсказка «сколько осталось до 2D» (2025-02-20)

- **Что сделано:** добавлен state `distanceTo2D` (число | null) и ref `lastDistUpdateRef` для throttle. В цикле `animate` при `!overlayShownRef.current` обновление `distanceTo2D` не чаще раз в 100 ms или при изменении `dist` не менее чем на 0.15. При переходе в 2D вызывается `setDistanceTo2D(null)`. В панели глобуса рядом с кнопкой «На карту» выводится подсказка: «До 2D: ~X% • ещё ~N приближ.» (X — процент от (dist−1.2)/(6−1.2), ограничен 100%; N — ceil(log(dist/1.2)/log(1.1))).
- **Ошибка:** подсказка исчезала после первого зума колёсиком.
- **Причина:** подсказка рендерилась только при `showGoToMapButton`, а кнопка «На карту» показывается при `dist >= 6`; после первого приближения dist становился < 6, кнопка и подсказка скрывались.
- **Решение:** показывать подсказку по условию `!showMapOverlay && distanceTo2D != null && distanceTo2D > MAP_THRESHOLD_ENTER_2D`, а не от `showGoToMapButton`. Тогда подсказка видна всё время на глобусе до перехода в 2D, независимо от того, видна ли кнопка «На карту».
