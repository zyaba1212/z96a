# Требования к проекту z96a

## Функциональные требования

### 1. Авторизация через Phantom Wallet
- Пользователь может подключить свой кошелёк Phantom.
- Подпись сообщения подтверждает владение кошельком.
- После авторизации появляются кнопки для работы с предложениями.

### 2. Главная страница с 3D-планетой Земля
- Отображение планеты в Full HD с возможностью приближения без потери качества.
- На планете чётко видны:
  - Границы государств и их названия.
  - Границы городов и их названия.
  - Существующая инфокоммуникационная сеть (кабели, оборудование, спутники, базовые станции и т.д.) с корректными обозначениями согласно официальным стандартам.
- Элементы управления:
  - Кнопка остановки/запуска вращения.
  - Кнопки приближения и отдаления.

### 3. Режим «Предложение» (доступен после авторизации)
- Кнопка «Предложить» ведёт на отдельную страницу редактирования.
- На странице отображается та же планета с текущей сетью (или последнее сохранённое предложение пользователя).
- Панель инструментов:
  - Удалить выделенный элемент.
  - Добавить элемент сети (выбор из предопределённого списка или создание нового элемента с указанием источника).
  - Соединить элементы (кабель, радиоканал).
- Все действия требуют подписания транзакции в кошельке.
- После подтверждения транзакции интерфейс обновляется.
- Ведётся история действий; можно откатить любое действие (также через транзакцию).

## Нефункциональные требования
- Производительность: плавное вращение планеты, отсутствие фризов при большом количестве объектов.
- Безопасность: все критические операции подтверждаются подписью пользователя.
- Масштабируемость: возможность добавления новых типов оборудования и элементов сети.
- Юзабилити: интуитивно понятный интерфейс, подсказки.
