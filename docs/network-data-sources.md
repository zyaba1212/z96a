# Источники данных сети (этап 4)

Данные о сети должны браться из **официальных источников** где возможно.

## Глобальная сеть (глобус и 2D)

- **Подводные кабели:** [TeleGeography Submarine Cable Map](https://www.telegeography.com/submarine-cable-map) — открытые данные о магистралях.
- **Спутники:** каталоги орбит (например CelesTrak, официальные реестры) — при добавлении реальных спутников.
- **Глобальные провайдеры:** названия и привязка к кабелям/узлам из публичных отчётов операторов.

## Локальная сеть (2D-карта)

- **Локальные провайдеры и оборудование:** данные с мест (региональные реестры, открытые карты операторов).
- На глобусе отображается только **глобальная** сеть; на 2D — глобальная + локальная (в т.ч. глобальные кабели видны и на карте).

## Обозначения (design.md)

- Иконки оборудования: ITU-T E.121, ГОСТ 2.721-74.
- Кабели: медь — оранжевый, оптика — зелёный.
