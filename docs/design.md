# Дизайн и визуализация

- Планета: текстура Earth (Blue Marble Next Generation) + бамп-карта.
- Границы государств: GeoJSON (Natural Earth Data).
- Названия: размещаются с помощью библиотеки для размещения текста на сфере.
- Иконки оборудования: использовать стандартные символы из ГОСТ/МСЭ (например, для мультиплексора — прямоугольник с крестом). Найти официальные источники: ITU-T Recommendation E.121, ГОСТ 2.721-74 и др.
- Кабели: линии разного цвета (медь — оранжевый, оптика — зелёный).
- Спутники: трёхмерные модели или спрайты с орбитами.
