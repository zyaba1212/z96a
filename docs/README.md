# Проект z96a

**Архитектура системы обработки транзакций для инфокоммуникационных систем в условиях частичного отсутствия сетевого соединения**

WEB3-приложение для визуализации и редактирования глобальной инфокоммуникационной сети на 3D-модели Земли с использованием кошелька Phantom (Solana).

## Основной стек
- Frontend: React + TypeScript, Next.js (или Vite), Three.js
- Web3: @solana/web3.js, @solana/wallet-adapter-react, Phantom Wallet
- Смарт-контракты: Rust + Anchor (Solana)
- База данных: PostgreSQL + Prisma (для хранения состояний сети)
- Стили: CSS Modules / Tailwind (на усмотрение)

## Запуск проекта
1. Клонировать репозиторий.
2. Установить зависимости: `npm install` (или `yarn`).
3. Настроить переменные окружения (создать `.env.local`).
4. Запустить в режиме разработки: `npm run dev`.
5. Открыть http://localhost:3000.

Подробнее в документации.
